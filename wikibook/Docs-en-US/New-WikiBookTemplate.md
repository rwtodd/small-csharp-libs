# New-WikiBookTemplate

## SYNOPSIS
Outputs a sample XML file to help you generate one.

## SYNTAX

```
New-WikiBookTemplate
```

## DESCRIPTION
An example XML file for a book is produced.

## EXAMPLES

### Example 1
```powershell
PS C:\> New-WikiBookTemplate > mytoc.xml
```

Put an XML template in the file `mytoc.xml`.
